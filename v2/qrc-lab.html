How to go Live & how this maps into the game
  1. Create an account at qurani.ai → Dashboard → get a QRC API key (this is where you'll also see pricing/subscription).
  2. Paste the key above, switch Mode to Live (QRC), pick the surah, press Recite.
  3. Confirm the audio format question in the yellow box; adjust transport.send() if the socket rejects chunks.
  4. Tune hafz_level / tajweed_level and the pass threshold so real kids almost never get wrongly blocked (watch the false-reject outcomes in the log).
  5. To wire into gate.js: each carved setting lights by reciting its ayah instead of dragging the gem; below threshold → play the ayah (hint) → after N tries the setting opens anyway (identical to the existing forgiving gate).
